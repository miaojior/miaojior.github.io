YMMC介绍

发表于 2021-08-15 阅读次数： 本文字数： 1.6k

野生技术协会——YMMC建模社

YMMC从2020年8月22日开始，在Minecraft中建模大连育明高级中学。此后，该项目逐渐演变成以MC为中心，研究各种硬核技术以完善建模项目的社群。2021年春季学期，我们申请成立了YMMC建模社。

随着科技发展，个人电脑性能逐渐提升，数字环境艺术也成为了每个人都有机会接触的领域。为了方便入门，我们选择了知名度较高且相对简单的Minecraft作为建模软件。Minecraft 是由Mojang Studios开发的沙盒游戏，其多样性、开放、高可自定义的特点，也极大降低了建模的门槛。所以不要害怕自己技术不够，加入我们，为最后的目标尽一份力，为育明多元化的宣传提供新的可能。

但是低准入门槛不代表能力底下。在其他学校已经拿出作品的情况下，后起之秀的我们如何做到有自己的特色，并在技术与艺术上全面超越呢？

首先就是收集数据，YMMC的建模过程中没有一块砖是凭感觉搭建的，我们主要通过参考学校工程图、实地测量的方式获取数据。测量过程中就将数据的国际单位转化为建模软件的单位，提前模拟冲突，补测细节，防止无效返工。因为学校依山而建，所以还需测量一项特殊的数据：高度。这部分用到了一些奇妙的物理数学知识进行计算，想了解更多就快点加入我们吧。

我们项目的最大优势在于软硬件支持。24小时稳定在线的服务器、定期保存快照和异地备份、包含多样模组的客户端等其他学校项目也都有，不值一提。YMMC真正独一无二的地方在于我们有自己开发的模组。这样就可以在Minecraft中实现单一墙壁内外材质不同、添加很多原版没有的材质、两个不同半砖出现在一格中、制作2米*3米的大门还一样能轻松开关等，原版MC无法实现的操作。由于软硬件支持的内容比较繁杂并需要一定计算机基础，这里不再赘述。想进一步了解的同学请阅读英文介绍。但请放心，学会如何使用它们是十分轻松的，并且以防出现问题，YMMC成员会不断收到客户端的功能更新和各种技术支持。

最后，想得到同学认可离不开宣传。这就要说说我们建造后的计划了。我们希望YMMC朝媒体向发展，培养出一个，从选题策划，文案撰写，分镜拍摄，到后期包装，审核投放，反馈总结，走完整流程戒骄戒躁的“职业”制片团队，在完工之后产出一些优质的数字媒体作品。我们已经做出一些尝试，这里举两个例子。

如果你已经看过技术展示部分的图片，你会发现它们与本页上的图片大不相同。以展示技术效果为目的，它只有最基本的原版材质，以及必要的简单构图。而本页的建成部分效果图，则是经过在MC中尝试各种光影，调整其参数，改变构图，从一批图片中精选出几张，并在PS里经过二级调色和风格化才达到现在的效果。

当你看到这篇文章时，YMMC的宣传视频应该已经发了。既然模型没有完成必不可能直接拍视频宣传，我们只能另寻他路。好在已经完成的部分截几张图做舞台背景还是足够的，再加上红石音乐，就有了社团才艺展示《国际歌》。其实，我们社团一直把宣传《国际歌》作为活动之一，这也是为什么YMMC的别名里有个Marxism。我们相信“全世界无产者联合起来，英特纳雄耐尔就一定要实现”，因此YMMC大部分的内容也进行了开源和共享。

欢迎每一个育明人加入YMMC，希望你能为我们带来新的技术，新的活力，让我们走向无限光明的未来。

A brief Introduction to the technical route of YMMC

发表于 2021-08-14 阅读次数： 本文字数： 4.1k

Minecraft players are often mistaken for tech experts, but this misunderstanding perfectly conforms to the situation here in YMMC because its members know not only how to play the game but also every aspect of modeling including measurement, building, and post-production. That is why YMMC, short for Yuming Minecraft, is joked as Yuming Measurement Marxism Mathematics & Music in Minecraft Club.

Now, How YMMC was established from the very beginning will be introduced below.

“Hello world”, which is known as the first lesson for programming language beginners, was sent to the server by miaojior on August 22th at 15:51:29. At first, the personal computers of YMMC members were used to run the server, but that was really a hassle since no one could be online all the time, especially after the new semester began. Then, an elastic compute service was rented so that the server could be available 24 hours a day as well as 365 days a year. To save money, the server only has a single-core CPU and two gigabytes of memory. Windows, of which the basic services and GUI occupy several gigabytes of RAM, is obviously not suitable for such a condition. Therefore, a terminal-only operating system like Linux that takes up only dozens of megabytes of memory to run its basic services spares us enough memory to run JVM.

What’s more, data security. Nobody wants to redo the work destroyed by data corruption. The only solution is to frequently back up the data, and if possible, add an allopatric backup. This solution is easier said than done but YMMC indeed implemented it. Our backups consist of three parts. First, the Alibaba elastic compute service is responsible for all real-time data calculations and saves all recent snapshots for quick recovery. In this way, if anything goes wrong, the server can immediately roll back to the previous snapshot. Second, when the project is partially completed, we will take a snapshot and send it back to the local storage for both quick recoveries in case of a server crash and recording every stage of the project. The local backups are stored on miaojior’s NAS and smartphone. Third, allopatric backup is aimed to provide an extra guarantee in case of unexpected events. The snapshots are monthly packaged and uploaded to computers at Corn High School， domestic and foreign cloud disks.

Minecraft Java Edition is adopted by YMMC. Because of Forge and Optifine, new features are easy to be added, but third-party mods do not satisfy all requirements and corn flavored blocks are impossible to be found. Fortunately, CRS decided to develop mods for YMMC. After months of developing and testing, it was released after the Opencorn Conference.

Opencorn 2020 Conference was held on February 10th, 2020, the day before New Year’s Eve. Engineer CRS spoke at the conference and released three mods aimed to make blocks smarter. The server and client entered a new era.

The first mod, myxrsidedblocks, provides blocks with surfaces made of different textures. For example, the exterior wall has bricks textured external surface and concrete textured internal surface but vanilla blocks are not able to meet the demand unless building two layers. Sided Block, the first type of myxrsidedblocks was introduced as a solution. However, after observing the campus, we found many structures that were not able to be built in Minecraft by both vanilla blocks and the newly introduced blocks, and more new types of blocks including Sided Slab that implements slabs with surfaces made of different textures, Triple Block that implements blocks with the two halves of top surface made of different textures and Window Edge that implements blocks to build the edge of exterior windows.

The second mod, Corndoors, implements door blocks more than 1 block in width and more than 2 blocks in height, and retractable gates. Since the scale of YMMC is one meter in reality to two blocks in Minecraft, the introduction of this mod enables us to build the gate and the doors of classrooms.

After the introduction of the mod Chisels & Bits that provides a way to sculpt and chip away at the individual cubes that make up many of the blocks, the colors of vanilla blocks no more meet our demand to build such blocks. Then, the mod Color216 that includes textures of 216 web safe colors was developed. This mod, apparently, is the least technical among all the three mods.

Then new versions of the client and server were announced for test and the game version was upgraded from 1.15.2 to 1.16.4. All of the features are working well, and YMMC members will continue receiving feature updates and technical support. To learn more about these mods, please visit GitHub and the official website. The source code of the mods is open-sourced under MIT and WTFPL.
