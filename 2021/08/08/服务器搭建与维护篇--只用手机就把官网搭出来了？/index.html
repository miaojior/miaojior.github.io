服务器搭建与维护篇--只用手机就把官网搭出来了？

发表于 2021-08-08 阅读次数： 本文字数： 1.8k

YMMC的官网 使用 hexo 和 next 主题
ps:正经用Windows搭建版请转至miaojior的博客（稍后附上链接）

前置知识

手机Android操作系统其实魔改自Linux，其内核是Linux内核。（前置知识的前置知识：Windows Mac Linux是PC的三大操作系统）所以只要有个终端手机就能当Linux用

关于Linux命令行的一些前置知识

其他Linux发行版 （termux用户请跳过此段）

本教程也适用于使用其他Linux发行版搭建网站，其中只有安装git和node部分略有不同，请参考各发行版使用的软件包管理器的说明及git和node官网教程
请确保node版本＞=12.0 一些发行版软件包管理器的node版本过于古老 你可能需要使用nodesource的二进制文件并配置环境变量来安装node

环境搭建

安装termux 并配置git和nodejs
termux二次开发版推荐
换源，安装zsh

apt update #更新软件包列表缓存
apt install -y git nodejs #安装git和node
npm config set registry http://registry.npm.taobao.org/ #将npm更换为淘宝源 如果您不是大陆用户 请忽略本步骤

安装hexo

mkdir hexo #创建工作目录
cd hexo #打开工作目录
npm install -g hexo-cli #安装hexo支持（全局）
hexo init #在工作目录初始化hexo
npm install #安装其他依赖
hexo g #生成静态网页
hexo s #启动hexo网页服务

打开localhost:4000 如果看到hexo欢迎界面 则安装成功

安装next主题

npm install hexo-theme-next #安装next主题的node依赖
git clone https://github.com/next-theme/hexo-theme-next themes/next #从github克隆整个next主题项目 放到theme文件夹中
apt install -y vim #安装vim
vim _config.yml #使用vim编辑器编辑

并用vim编辑器将 theme 后面的landscape 改为next
vim的使用
再次运行

hexo g
hexo s

浏览器打开localhost:4000 主题变了则切换成功

更改工作目录下的_config.yml文件中的一些配置

vim _config.yml

编辑 #site 部分的信息

hexo g
hexo s

浏览器中再次打开localhost:4000 标题等内容就会发生改变

# YMMC官网使用的配置
# Site
title: YMMC
subtitle: 'Yuming Minecraft Club'
description: 'Yuming Minecraft Club'
keywords:
author: YMMC
language: zh-CN
timezone: 'Asia/Shanghai'

写作

hexo 默认使用 markdown 渲染网页
你需要一个markdown编辑器来写作
请自行选择
markdown语法

发表

hexo new post "title" #新建以title为名的文章 存放在 ./source/_post/title.md
vim ./source/_post/title.md

更改标题等信息，将写好的内容粘贴到横线以下部分，保存并退出

hexo g
hexo s

再次在浏览器中打开localhost:4000你就可以看到自己的文章了
