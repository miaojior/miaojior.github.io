关于命令行（以zsh为例）从左至右介绍

username@hostname

代表了用户@所使用的机器名

目录

Linux的一些特殊目录

  • / 为根目录
  • ~为用户目录
  • .为当前目录
  • ..为上一级目录 根目录没有上一级目录 废话

如何表示一个目录

如 用户目录下的new目录 ~/new
如 根目录下的old目录 /old
如 new的套娃表示 ~/new/../new/./ #用户目录的new目录的上一级目录的new目录的当前目录
当前目录可省略不写 比如当前目录就是 ~/ 那么new 就等价于 ~/new
你可以学习完cd命令后尝试本部分内容
注意特殊目录在此处的使用

如何表示一个文件

如 new.txt #表示当前目录下的new.txt文件
同理 你也可以用目录加文件名的方式
如 ~/new/new.txt

输入区

在此处输入命令 按回车执行
终端上会返回执行结果
初学者建议逐条执行命令 不要大段复制
注：本网站上以#为注释符号 即同一行#后的部分为对命令的解释 不需要执行（简单来说英文部分才是真正的命令）
命令运行结束时 回自动重新打开下一个命令行 一些不会自己结束的命令 按Ctrl+C强制停止

Linux的一些基本命令（建议同时打开终端尝试）

  • file 可以用任何表示文件的方式表示
  • directory 可以用任何表示目录的方式表示

list

ls  #列出当前目录下的文件及目录
#ls 命令不会列出以.开头的隐藏文件 如.和.. 如果需要全部显示 请使用
ls -a

make directory

mkdir documents  #新建以"documents"为名称的文件夹

change directory

cd directory #进入一个目录 即打开一个文件夹
#一些特殊的cd
cd documents #进入当前目录下的documents目录
cd .. #返回上一级目录
#当然你也可以在 documents目录原地传送
cd .././documents

copy

cp source_file directory #将源文件文件复制到 目标目录下
cp -r source_directory directory #源目录整个复制到目标目录下
# 当源为目录时 需要添加-r参数对整个目录递归操作
cp source_file file_backup #在当前目录下创建源文件的副本

move

mv source_file directory #将源文件移动到目标目录下
mv source_directory directory #将源目录移动到目标目录下
#你是不是在想为什么没有重命名 的确 Linux没有 rename这种指令 因为你只需要 mv
mv file1 file2 #将file1重命名为 file2

remove

rm file #删除文件
rm -rf direcroty #删除目录和目录下所有文件
#请注意：在linux命令行下操作，没有回收站这东西给你后悔用
